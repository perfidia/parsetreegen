Parse Tree Generator
====================

Description
-----------

Simple and small tool that allows everyone to generate syntax diagrams in SVG format.

Installation
------------

### Simple

    python setup.py install

### Using eggs

    python setup.py bdist_egg
    cd dist
    easy_install <package_name>

Getting started
---------------

See main.py file.
